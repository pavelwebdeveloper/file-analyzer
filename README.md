# Project Title

File Analyzer

## Instructions for Build and Use

[Software Demo Video](https://youtu.be/QdBPt0DW51c)

Steps to build and/or run the software:

1. Install Code::Blocks IDE on your computer (you might also use either Eclipse or Visual Studio instead of Code::Blocks)
2. Run "git clone https://github.com/pavelwebdeveloper/file-analyzer.git" locally on your computer in CMD
3. Build and run the code in IDE (I used Code::Blocks)

Instructions for using the software:

1. Build and run the program in IDE 
2. A window appears with a question about the name of a file to analyze (I provided a test file in the repository which you can use or you can provide your own text file).
3. Continue to answer the questions that appear in the screen.
4. After a successful execution of the program you can find a new file in the root directory of the program with the name that you specify in answer to one of the questions.
